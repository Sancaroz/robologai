# RoboLogAI Terminal Cheatsheet

## Open project
```bash
cd ~/Desktop/ROBOLOGAI
```

## Validate data
```bash
node scripts/validate-data.mjs
```

## Validate assets
```bash
node scripts/validate-assets.mjs
```

## Dry-run build
```bash
node scripts/build-data.mjs
```

## Full write/build
```bash
node scripts/health-check.mjs --write
```

## Git status
```bash
git status
```

## Commit changes
```bash
git add .
git commit -m "message"
git push
```
